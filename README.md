### Run Locally

To run **DeliDelight** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/sumit-saha-x/DeliDelight.git
```

Windows:

```bash
git clone https://github.com/sumit-saha-x/DeliDelight.git
```
